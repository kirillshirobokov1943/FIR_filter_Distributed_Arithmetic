Описание КИХ-фильтра с построянными коэфициентами, реализованного на рпспределеной арифметике (FPGA-реализация)
Параметры фильтра:
глубина фильтра: 16
формат входных данных:  signed Q1.15
формат выходных данных: signed Q1.15
формат коэфициентов фильтра: signed Q1.15
коэфициенты задаются как параметры (по умолчанию - коэфциенты ФНЧ с полосой пропускания [0;fs/4], 
вычисленные оконным методом с использованием окна Кайзера с параметром betta = 5.65326)

16 входных отсчетов x_in параллельно загружаются на входы x_in[0], ... x_in[15]. Вместе с отсчетами  подаётся сигнал vld_in - сигнал валидности входных данных. 
Спустя 13 тактов после принятия входных отсчетов x[i], ... x[i+15] модуль выдает выходной отсчет y[i+15] на выходе y_out вместе с сигналом vld_out.
В течение 13 тактов после загрузки входных отсчетов на модуль нельзя подавать следующие входные данные с сигналом vld_in.
