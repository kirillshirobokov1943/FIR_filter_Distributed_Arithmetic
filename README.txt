Описание КИХ-фильтра с построянными коэфициентами, реализованного на распределеной арифметике (FPGA-реализация)
Параметры фильтра:
глубина фильтра: 16
формат входных данных:  signed Q1.15
формат выходных данных: signed Q1.15
формат коэфициентов фильтра: signed Q1.15
коэфициенты задаются как параметры (по умолчанию - коэфциенты ФНЧ с полосой пропускания [0;fs/4], 
вычисленные оконным методом с использованием окна Кайзера с параметром betta = 5.65326)

16 входных отсчетов x_in параллельно загружаются на входы x_in[0], ... x_in[15]. Вместе с отсчетами  подаётся сигнал vld_in - сигнал валидности входных данных. 
Спустя 13 тактов после принятия входных отсчетов x[i], ... x[i+15] модуль выдает выходной отсчет y[i+15] на выходе y_out вместе с сигналом vld_out.
В течение 13 тактов после загрузки входных отсчетов на модуль нельзя подавать следующие входные данные с сигналом vld_in.

Описание файловой структуры проекта:
block diagrams - папка со структурными моделями дизайна, математической основой его работы
Matlab_model_filtering.m - Matlab скрипт с моделью фильтра и результатами работы фильтра (в кчаестве входных воздейтсвий - сумма синусусоид). Сохраняет входные и выходные отсчеты в файлы x.txt и y.txt соответственно  
Matlab_model_filtering.fig - рузультаты работы модели фильтра (графики входных и выходных отсчетов)
FIR_filter_DA.sv, sp_bl.sv - файлы дизайна
tb.sv - тестбенч, симулирующий работы дизайна и сохраняющий валидные выходные отсчеты в файл y_out.txt (временные диаграммы неинформативны, т.к. данные на выходе корректны только с сигналом vld_out)
run_sim - скрипт для запуска тестбенча
y_out_graph.m - скрипт для построения результатов работы дизайна (графики входных и выходных отсчетов, выходные отсчеты начинаются с 15-ого индекса)
DA_filtering.fig - рузультаты работы дизайна (графики входных и выходных отсчетов, выходные отсчеты начинаются с 15-ого индекса)
